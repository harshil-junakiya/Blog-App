# FULL-Stack Blog App

## How To Run
Create an Atlas URI connection parameter in `backend/.env` with your Atlas URI:
```
MONGO_URL=mongodb+srv://<username>:<password>@sandbox.jadwj.mongodb.net/myFirstDatabase?retryWrites=true&w=majority
PORT=4000
```

Start server:
```
cd backend
npm install
npm start
```

Start Web server
```e
cd frontend
npm install
npm run server
```
